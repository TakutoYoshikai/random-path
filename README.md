# random-path
This is a node package to generate path to directory randomly.

### Requirements
* Linux
* Node.js

### Usage
**install**
```bash
npm install -g TakutoYoshikai/random-path
```

**generate a path**
```bash
random-path <DIR>
```

**use in JavaScript**
```javascript
const RandomPath = require("random-path");
console.log(RandomPath("<DIR>"));
```
